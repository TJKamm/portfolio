Web Development Resources

Front-End Coding

  • W3 Schools:

    The largest web development site, with a plethora of tutorials and examples for HTML, CSS, Javascript, and more.

  • Codecademy:

    A wide variety of coding tutorials that allow you to practice right on the page, alongside the instructions.

  • CSS-Tricks:

    An extensive blog encompassing all things CSS.

  • Formatting & Indenting Your HTML:

    An article by Scott Granneman detailing how to properly format HTML code.

  • Bootstrap:

    CSS, Javascript, and fonts all minified and compiled for simple use with your site's HTML.

  • HTML Validator:

    A useful tool for ensuring your HTML code is valid.

  • Boilerplate:

    A front-end template for building a robust site quickly.

  • FontAwesome:

    Font and CSS toolkit that includes icons which can be easily customized for size, color, and anything else that can be changed with CSS.

Web Design

  • WebDesign.org:

    Tutorials, articles, and more.

  • Dribbble:

    A site that showcases a vast array of art and graphics, perfect for design inspiration.

  • Adobe Color:

    A useful tool for finding the perfect color theme for your site.

  • Favicon:

    A tool that allows you to create and download a Favicon for your site.

Other Links

    An open-source version control tool for collaboration and web hosting.

  • Lynda.com:

    A site with tutorials on everything from Web Development and Design to Photoshop and Business Skills.
